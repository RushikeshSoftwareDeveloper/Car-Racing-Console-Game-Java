# Car-Racing-Console-Game-Java
Console-based Car Racing Game developed in Java where players avoid obstacles using left and right movement.
# 🚗 Car Racing Game (Console Version - Java)

## 📌 Project Description
A simple console-based Car Racing simulation developed using Core Java.
The player controls a car and must avoid incoming obstacles by moving left or right on the road.

## 🚀 Features
- Player car movement (Left / Right)
- Random obstacle generation
- Collision detection
- Game Over condition
- Console-based game simulation

## 🛠 Concepts Used
✔ Loops (Game loop simulation)  
✔ Arrays (road representation)  
✔ Random class (obstacle generation)  
✔ Conditional logic  
✔ Basic Object-Oriented Programming

## 🎮 Game Controls
| Key | Action |
|----|----|
| A | Move Left |
| D | Move Right |

## ▶ How to Run
1. Clone the repository
2. Open the project in Eclipse / IntelliJ
3. Run the main class
